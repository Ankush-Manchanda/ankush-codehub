## 🛠 How to Use This Portfolio

- Built using **ReactJS**
- Fully customizable by editing one file: `src/portfolio.js`
- Auto-deployed using **GitHub Pages**

### 📝 Update Your Info

Open `src/portfolio.js` and edit:

- `greeting` — name, title, subtitle
- `socialMediaLinks` — GitHub, LinkedIn, Twitter, etc.
- `skills`, `degrees`, `experience`, `certifications`, `projectsHeader`, `contactPageData`

---

## 🌈 Change Theme

Go to `src/theme.js`  
Update this line:

```js
export const chosenTheme = blueTheme;
```

Replace `blueTheme` with your preferred theme like `darkTheme`, `oceanBlueTheme`, etc.

---

## 🚀 Deploy on GitHub Pages (No Terminal Required)

1. Go to your GitHub repository
2. Click on **Settings → Pages**
3. Under **Source**, choose:
   - **Branch**: `main` or `gh-pages`
   - **Folder**: `/root`
4. Click **Save**
5. Your portfolio will be live at:  
   `https://ankush-manchanda.github.io/ankush-codehub/`
